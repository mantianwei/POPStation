# vue-koa-mysql—demo

> 刚接触全栈小同学为POPStation线上H5项目做的第一个demo，项目刚刚起步，还没有整合使用vue，myaql。目前初步搭好了koa后台设置路由，基于jquery动效库以及现有动效，结合POPStation需求做了两个演示动效，接下来主要工作是以上线为目标开发H5结构，根据需要使用vue实现交互原型，完成粉丝留言领劵的功能。

## 项目说明

- UI：暂定
- 前端框架：Vue（还未整合）
- 后端框架：Koa
- 数据库：Mysql（还未加入）
- 部署：PM2
